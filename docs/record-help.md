# 처음 시작하기

처음부터 완벽한 기록을 만들 필요는 없다.

강의자료 하나만 있어도 시작할 수 있다. 예전에 AI와 공부했던 대화, 시험 정리 노트, 필기는 나중에 더하면 된다.

## 기본 흐름

```text
자료 넣기 -> 기록에 정리하기 -> 다시 질문하기 -> 복습할 것 남기기 -> 다음 공부에 이어 쓰기
```

## 폴더

```text
raw/   아직 정리되기 전의 자료를 넣는 곳
notes/  정리된 학습 기록이 쌓이는 곳
docs/  사용 방법을 다시 볼 수 있는 곳
```

## 무엇부터 시작할까?

`/record help`를 실행하면 아래 선택지로 시작한다.

```text
1. 강의자료 PDF가 있어요.
2. 예전에 AI와 공부했던 대화가 있어요.
3. 필기나 시험 정리 노트가 있어요.
4. 과목 전체를 천천히 옮기고 싶어요.
5. 이미 쌓인 기록에 질문하고 싶어요.
```

각 선택지는 필요한 방법만 안내한다. 처음부터 모든 명령을 외울 필요는 없다.
