# Obsidian으로 열기

Sillok은 기본적으로 Markdown 폴더입니다. Obsidian을 쓰면 링크, 검색, 그래프 보기를 더 편하게 사용할 수 있지만 필수는 아닙니다.

## 열기

1. Obsidian을 설치합니다.
2. Obsidian을 실행합니다.
3. `Open folder as vault`를 선택합니다.
4. `sillok` 폴더를 선택합니다.

## 처음 볼 파일

먼저 아래 파일을 열어보면 충분합니다.

- `notes/index.md`
- `notes/questions.md`
- `notes/review.md`

## 플러그인

처음에는 아무 플러그인도 필요 없습니다.

나중에 필요해지면 아래 정도만 검토하세요.

- Git
- Dataview
- Omnisearch

처음부터 플러그인을 많이 설치하지 않아도 됩니다. Sillok은 평범한 Markdown 폴더로도 작동해야 합니다.
