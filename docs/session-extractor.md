# AI 대화에서 공부 내용 꺼내기

이 프롬프트는 ChatGPT, Claude, Gemini 같은 기존 AI 대화 끝에 붙여넣기 위한 것이다.

목적은 대화를 예쁘게 요약하는 것이 아니다. 나중에 같은 주제를 다시 공부하거나 작업할 때 이어서 쓸 수 있는 재료를 꺼내는 것이다.

## 붙여넣을 프롬프트

```text
이 대화를 나중에 다시 이어서 공부하거나 작업할 때 참고할 수 있도록 정리해줘.

단순 요약이 아니라, 내가 이 대화를 통해 무엇을 이해했고 무엇을 다음에 다시 확인해야 하는지 중심으로 정리해줘.

정리할 때 지켜줘.

- 대화 전체를 빠짐없이 요약하려고 하지 마.
- 새 교과서 설명을 쓰지 마.
- 내가 이미 이해한 내용과 아직 다시 확인해야 할 내용을 나눠줘.
- 대화 중 내 이해가 바뀐 부분이나 도움이 됐던 설명을 살려줘.
- 내가 직접 말한 표현, 예시, 비유가 중요하면 그대로 남겨줘.
- 확실하지 않은 내용은 단정하지 말고, 확인이 필요하다고 표시해줘.
- 다음에 이 주제로 다시 질문할 때 어떤 내용을 전제로 설명해도 되는지 알려줘.
- 반대로, 어떤 내용은 건너뛰지 말고 차근차근 설명해야 하는지도 알려줘.

출력은 아래 흐름으로 정리해줘.

# 세션 정리

## 이 대화에서 다룬 것
처음에 내가 무엇을 이해하려고 했는지, 어떤 문제를 해결하려고 했는지 짧게 정리해줘.

## 내가 이해한 내용
이 대화를 통해 내가 이해했다고 볼 수 있는 내용을 정리해줘.
다음에 같은 주제를 다룰 때 너무 기초부터 반복하지 않아도 되는 내용만 적어줘.

## 다시 확인하면 좋은 내용
아직 내가 정확히 설명하기 어렵거나, 비슷한 개념과 구분해서 다시 봐야 할 내용을 정리해줘.

## 도움이 된 설명
나에게 특히 도움이 됐던 설명, 예시, 비유, 비교가 있다면 남겨줘.

## 다음 대화를 위한 설명 방식
다음에 이 주제로 설명할 때 참고할 수 있도록 적어줘.

- 이미 알고 있다고 봐도 되는 내용
- 천천히 이어서 설명해야 하는 내용
- 사용하면 좋은 예시나 비유
- 피하면 좋은 설명 방식

## 연결되는 개념
이 대화가 연결되는 과목, 개념, 프로젝트, 경험을 적어줘.
억지로 많이 만들지 말고 실제로 의미 있는 연결만 남겨줘.

## 다음에 물어보면 좋은 질문
이 주제를 이어서 공부하거나 작업할 때 도움이 될 질문을 3~7개 적어줘.

## 한 문장 요약
이 대화를 한 문장으로 정리해줘.

형식보다 내용이 중요해.
나중에 다시 읽었을 때 "내가 이 대화에서 무엇을 얻었고, 다음에 무엇을 이어서 보면 되는지"가 바로 보이게 써줘.
```
