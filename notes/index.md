# Sillok Notes

여기는 학습 기록이 쌓이는 공간입니다.

처음 시작한다면 `/record help`를 사용하세요.

## Start Here

- [[questions]] - 아직 남아 있는 질문
- [[review]] - 다시 볼 것
- [[_stubs]] - 나중에 만들면 좋은 페이지
- [[log]] - 에이전트가 바꾼 기록

## Courses

과목, 책, 강의 시리즈처럼 이어지는 학습은 여기에 추가됩니다.

과목 전체를 천천히 옮길 때는 먼저 계획을 세웁니다.

```text
/record course backfill <course-slug> --plan
```

## Concepts

공부하면서 반복해서 쓰일 개념 페이지가 생기면 여기에 연결됩니다.

## How To Use

- `/record help` - 처음 시작하거나 다음 행동을 고를 때
- `/record study` - 단발 공부를 기록에 넣을 때
- `/record course` - 이어지는 학습을 관리할 때
- `/record query` - 이미 쌓인 기록에 질문할 때
