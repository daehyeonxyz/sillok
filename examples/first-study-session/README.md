# 첫 공부 세션 예시

이 예시는 작은 공부 기록 하나가 Sillok에서 어떻게 기록 재료가 되는지 보여준다.

## 해보기

1. `raw-note.md`를 읽는다.
2. 에이전트에게 아래처럼 요청한다.

```text
/record study examples/first-study-session/raw-note.md
```

3. 에이전트는 상황에 따라 아래를 갱신한다.

- session/source record
- 필요한 concept page
- `notes/questions.md`
- `notes/review.md`
- `notes/log.md`

## 볼 점

목표는 원본 노트를 그대로 복사하는 것이 아니다.

목표는 다음 공부에 영향을 줄 질문, 복습 지점, 이해한 내용을 남기는 것이다.
